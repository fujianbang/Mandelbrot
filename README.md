# Mandelbrot Set

This is a simple implementation of the Mandelbrot in Rust

## Usage

```bash
cargo build --release
.\target\release\mandelbrot.exe mandel.png 4000x3000 -1.20,0.35 -1,0.20
```

You can also use the `cargo run` command to run the program directly.
